== Description ==

Import <a href="https://wpbstone.com/demos/" target="_blank" rel="nofollow">Bstone</a> theme demo content, widgets and theme settings with just one click.

Get [free support](https://wpbstone.com/contact-us/)

= Requirements =

* WordPress 4.7 or later.
* [Bstone Theme](https://wpbstone.com/).

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the Bstone Demo Importer plugin through the 'Plugins' menu in WordPress.

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of Bstone Demo Importer, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Bstone Demo Importer” and click Search Plugins. Once you’ve found our demo importer plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading our demo importer plugin and uploading it to your webserver via your favourite FTP application. The WordPress codex contains [instructions on how to do this here](https://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
